# 情緒分析

YouTube是其中一個最具影響力的平台，可影響消費者的認知和品牌聲譽。 當AI系統回應有關您品牌的提示時，他們越來越多地引用YouTube影片作為來源，使得在該內容中討論您的品牌的方式直接輸入到AI產生的回應中。

在您的情緒儀表板提示集中偵測到YouTube影片作為提示引文時，就會出現YouTube品牌存在感分析機會。 它會分析這些被引述的影片及其對於情緒、聲音份額和常見主題的評論。 然後它會顯示優先建議，以改進在AI產生的回應中如何看待和表示您的品牌。

它會透過六個維度分析您的品牌：

- **已分析的視訊** — 已檢查品牌提及和情緒的YouTube視訊數量。
- **已分析評論** — 已分析視訊中檢查的評論數目。
- **品牌提及（影片）** — 在影片內容中提及您品牌的頻率。
- **品牌提及（註解）** — 評論中提及您品牌的頻率。
- **整體情緒（影片）** — 跨影片內容呈現您品牌的彙總情緒。
- **整體情緒（評論）** — 評論間對您品牌的彙總情緒。

>[!NOTE]
>情緒分析目前仍在測試階段。 功能與可用性可能會隨著功能的持續開發而改變。

![情緒分析儀表板](/help/dashboards/opportunities/assets/youtube-sentiment-overview.png)

## 運作方式

LLM Optimizer會監視AI系統所引用的YouTube影片，以在品牌存在感儀表板提示集中顯示提示。 當偵測到引用的影片時，它會分析這些影片及其對品牌提及、情緒、聲音份額和AI引文的評論。 它會比較您的品牌與市場競爭對手及相關品牌的效能、識別推動情緒的週期性主題，並產生建議以解決認知差距。

如果沒有為您提示集中的提示引發YouTube影片，此機會將不會顯示在您的儀表板中。

結果會顯示在兩個索引標籤中： **建議**&#x200B;和&#x200B;**效能**。

## 建議

此標籤顯示改善品牌在YouTube上的認知度的建議。 建議會整理成三個子標籤： **目前的建議**、**已修正的建議**&#x200B;以及&#x200B;**忽略的建議**。

![建議索引標籤](/help/dashboards/opportunities/assets/youtube-sentiment-suggestions.png)

建議表格包含下列各欄：

- **建議** — 建議的改善專案，以解決認知差距。
- **優先順序** — 緊急等級（嚴重、高、Medium、低）。
- **動作專案** — 開啟包含實施建議之特定步驟的面板，包括負責團隊（例如，內容策略、影響者行銷、產品行銷）。
- **證據** — 開啟「來源」表格，顯示建議背後的影片。

展開建議會顯示&#x200B;**AI分析**&#x200B;區段，其中包含：

- **為什麼需要改進** — 說明已發現的認知差距，包括競爭環境以及問題在YouTube內容中的形成方式。
- **如何改善** — 關於哪些內容或動作可以填補空白的特定指引。
- **預期結果** — 實作建議的預期結果。

**來源**&#x200B;表格顯示推動建議的YouTube影片，包含下列欄：

- **影片** — YouTube影片的標題和連結。
- **頻道** — 發佈視訊的YouTube頻道。
- **參與度** — 參與度等級（低、Medium、高）。
- **品牌提及** — 您的品牌提及數與影片中提及的總數。
- **聲音份額** — 您品牌相對於所有提及品牌的提及份額。
- **前5名品牌** — 影片中提及次數最多的品牌。
- **情緒** — 影片中您品牌的整體情緒。
- **AI引用** — 引用此影片的AI答案數。

## 績效

**效能**&#x200B;索引標籤提供您品牌在YouTube內容中的表現詳細劃分。 它分為四個區段。

### 市場前景

根據提及內容比較您的品牌與相關品牌和市場競爭對手的績效。

![市場橫向](/help/dashboards/opportunities/assets/youtube-sentiment-market-landscape.png)

它顯示：

- **影片中的品牌提及** — 您的聲音份額與相關品牌和市場競爭對手的比較。
- 註解中的&#x200B;**品牌提及** — 各註解內容之間的比較是相同的。
- **市場追蹤** — 可篩選的圖表，最多可選取五個競爭者品牌，比較各影片和評論的聲音份額。

### 情緒分析

使用&#x200B;**情緒分佈**&#x200B;圖表追蹤分析內容中的品牌認知，該圖表顯示對視訊和評論有利的、中立的和不利的情緒百分比劃分。

![情緒分析](/help/dashboards/opportunities/assets/youtube-sentiment-distribution.png)

### 影片

已分析YouTube影片的詳細表格，包含下列欄：

- **影片** — YouTube影片的標題和連結。
- **頻道** — 發佈視訊的YouTube頻道。
- **參與度** — 參與度等級（低、Medium、高）。
- **品牌提及** — 您的品牌提及數與影片中提及的總數。
- **聲音份額** — 您品牌相對於所有提及品牌的提及份額。
- **前5名品牌** — 影片中提及次數最多的品牌。
- **情緒** — 影片中您品牌的整體情緒。
- **AI引文** — 與視訊相關聯的AI引文訊號數目。

[效能]索引標籤會以單一檢視顯示&#x200B;**影片**&#x200B;和&#x200B;**主題**&#x200B;面板（已選取&#x200B;**影片**）。 下圖包含視訊層級表格，其下方為&#x200B;**主題**&#x200B;摘要。

### 評論

已分析YouTube註解的詳細表格，其中包含與「視訊」表格相同的欄，經篩選後會成為註解層級的資料。

### 主題

在分析內容中識別的週期性主題的表格，顯示：

- **主題** — 已識別週期性主題或主旨。
- **品牌提及** — 與主題相關聯的品牌提及數目。
- **情緒** — 與主題相關的整體情緒。

**主題**&#x200B;表格與「視訊」表格出現在相同的「效能」檢視中；請參閱上面[視訊](#videos)區段中的圖。

## 常見問題

**為什麼YouTube對AI 搜尋很重要？**

AI系統在產生有關品牌、產品和主題的回應時，越來越多地引用YouTube影片。 當這些引用影片以不利或不準確的方式討論您的品牌時，該情緒會直接進入AI系統如何代表您的品牌。 改善AI系統所引述的YouTube內容中討論您品牌的方式，是影響AI產生品牌認知的最直接方式之一。

**為什麼這個機會沒有顯示在我的儀表板中？**

只有在偵測到YouTube影片作為品牌存在感儀表板提示集中的提示引文時，才會出現此機會。 如果沒有為這些提示引用YouTube影片，則不會顯示機會。 當您的品牌獲得更多的YouTube涵蓋範圍，並且這些影片被AI系統引用於提示集時，機會將可供使用。

**整體情緒代表什麼？**

整體情緒反映了提及您品牌時內容的整體語調：有利、中立或不利。 視訊和評論會分別計算平均資訊量，因為兩者可能大不相同。

**什麼是聲音份額？**

聲音份額是指您的品牌在指定內容內或所有分析內容中，相對於其他所有提及品牌的總體品牌提及百分比。

**什麼是AI引文？**

AI引文顯示有多少個AI答案引用了給定的影片。 較高的AI引文數表示AI系統在產生相關主題的回應時主動使用視訊，使得這些視訊中的情緒對於您品牌的AI代表性特別重要。

**如何識別市場競爭者？**

系統會根據您品牌的產業以及在分析內容中最常共同提及的品牌，自動識別競爭者。 您也可以手動選取最多5個品牌，以便在市場追蹤圖表中比較。

**分析多久更新一次？**

YouTube分析會反映分析處理結果，顯示控制面板標題中顯示的日期。 實施建議後重新造訪商機，以追蹤情緒和聲音份額中的變更。

# 新增目錄

>[!NOTE]
>
> **提早存取** — 此功能可在「提早存取」中使用。 隨著功能日漸成熟，工作流程的可用性、資格和部分可能會有所變更。 如果您對存取權有任何疑問，請聯絡您的Adobe帳戶團隊。

新增目錄機會會識別缺少清晰&#x200B;**目錄**&#x200B;和結構指南的高流量頁面，這會讓AI代理程式更難以剖析頁面並將使用者查詢對應到正確的區段。 它引進了結構化的目錄，其中包含&#x200B;**錨點連結的標題**，可反映頁面的主要區段。 此結構有助於代理程式更可靠地擷取、對應和引用相關段落。

對於每個受影響的URL，您可以檢閱建議的目錄專案，然後透過[在Edge](/help/dashboards/optimize-at-edge/overview.md)最佳化來部署這些專案，這樣代理流量就可以接收更清楚的導覽內容，而不需要任何內容管理系統(CMS)變更。

## 如何修正問題

使用[在Edge](/help/dashboards/optimize-at-edge/overview.md)最佳化(Optimize)套用修正，其中：

- 向AI代理程式提供預先轉譯的HTML快照。
- 將目錄新增至頁面。
- 適用於CDN層（CMS不會變更）。
- 僅限AI — 不會對人類訪客或SEO機器人造成影響。
- 幾分鐘即可部署，而且可從LLM Optimizer介面&#x200B;**完全還原**。

## 運作方式

LLM Optimizer會識別&#x200B;**目錄**&#x200B;可改善AI代理程式導覽標題和區段的高流量頁面。 對於每個頁面，建議均以頁面上已存在的標題為基礎，因此目錄會反映頁面結構。

受影響的URL會出現在&#x200B;**目前的建議**&#x200B;標籤上含建議&#x200B;**的** URL中。

在&#x200B;**目前的建議**&#x200B;中，您可以針對每個URL：

- **展開列**&#x200B;以檢查建議的目錄（從頁面上的標題剖析，並以錨點連結的專案呈現）。
- **比較前後預覽**。
- 如果您在LLM Optimizer外部處理商機，請&#x200B;**標示為已修正**。
- **忽略不相關的**&#x200B;建議。

建議會整理為&#x200B;**目前的建議**、**固定的建議**&#x200B;和&#x200B;**忽略的建議**，這與Edge提供的其他最佳化機會一致。

### 部署最佳化

當您準備在邊緣發佈時，請選取您要部署的目錄建議。 頁尾總結選取的專案數，通常會提供&#x200B;**標籤為固定**、**忽略建議**&#x200B;和&#x200B;**部署最佳化**。

按一下&#x200B;**部署最佳化**。 **部署至Edge**&#x200B;對話方塊會列出選取的URL和最佳化詳細資料。 檢閱清單，然後選擇&#x200B;**部署**&#x200B;或&#x200B;**取消**。

成功部署後，**部署完成**&#x200B;會確認有多少最佳化已上線。 關閉對話方塊並開啟&#x200B;**修正建議**&#x200B;以驗證狀態。

>[!NOTE]
>
>部署最佳化需要在Edge上線流程中完成最佳化。 如果您尚未上線，按一下&#x200B;**部署最佳化**&#x200B;會將您導向到上線流程。 如需Edge最佳化如何運作、支援的CDN提供者和上線流程的完整詳細資訊，請參閱[Edge最佳化](/help/dashboards/optimize-at-edge/overview.md)頁面。

### 修正建議和即時檢視

在&#x200B;**已修正的建議**&#x200B;上，部署的URL在狀態列中顯示&#x200B;**已最佳化**。 展開資料列以檢閱已部署的目錄、使用&#x200B;**詳細資料**&#x200B;以取得可用的分析，或按一下&#x200B;**即時檢視**&#x200B;以開啟供驗證的&#x200B;**目前頁面內容**&#x200B;的唯讀檢視（包括插入的&#x200B;**目錄**）。

當您需要大量回覆邊緣變更時，請使用核取方塊選取最佳化的列，然後在標題中使用&#x200B;**Rollback**。

## 復原

如果您改變心意，可以復原已部署的最佳化。 從&#x200B;**修正建議**&#x200B;檢視中，選取您要回覆的最佳化資料列，然後按一下標頭中的&#x200B;**回覆**。

**回覆**&#x200B;對話方塊會列出要回覆的建議，並附上簡短的警告，說明部署的最佳化將會回覆。 確認清單，然後按一下&#x200B;**回覆**&#x200B;或&#x200B;**取消**。

作業完成後，會顯示&#x200B;**已成功回覆**&#x200B;摘要；請關閉摘要以返回儀表板。

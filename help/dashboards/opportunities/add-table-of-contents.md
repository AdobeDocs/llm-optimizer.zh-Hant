# 新增目錄

>[!NOTE]
>
> **搶先體驗**：此功能目前處於搶先體驗階段。 隨著此功能逐漸成熟，其可用性、適用資格及部分工作流程可能會有所變更。 如果您對存取權有任何疑問，請聯絡您的 Adobe 帳戶團隊。

「新增目錄」機會可識別缺少清晰&#x200B;**目錄**&#x200B;和結構指南的高流量頁面，這類頁面讓 AI 代理難以剖析頁面內容及將使用者查詢對應至正確區段。 此機會引進結構化目錄，其中包含反映頁面主要區段的&#x200B;**錨點連結標題**。 此結構有助於 AI 代理更可靠地擷取、對應和引用相關段落。

針對每個受影響的 URL，您可以審閱建議的目錄項目，然後透過[邊緣最佳化](/help/dashboards/optimize-at-edge/overview.md)進行部署，讓代理式流量獲得更清晰的導覽情境，且無需變更內容管理系統 (CMS)。

## 其修正問題的方式

將透過[邊緣最佳化](/help/dashboards/optimize-at-edge/overview.md)套用修正，且具備以下特點：

- 為 AI 代理提供預先轉譯的 HTML 快照。
- 在頁面中新增目錄。
- 在內容傳遞網路層運作 (無需變更 CMS)。
- 僅針對 AI 運作，不會影響真人訪客或搜尋引擎最佳化機器人。
- 只需幾分鐘即完成部署，且可透過 LLM Optimizer 介面&#x200B;**完全還原**。

## 運作方式

LLM Optimizer 會識別可透過&#x200B;**目錄**&#x200B;改善 AI 代理導覽標題和區段之能力的高流量頁面。 對於每個頁面，建議皆以頁面上的既有標題為基礎，以便目錄能反映頁面結構。

受影響的 URL 會顯示在&#x200B;**「目前建議」**&#x200B;標籤的&#x200B;**「有建議項目的 URL」**&#x200B;表格中。

在&#x200B;**「目前建議」**&#x200B;中，您可以針對每個 URL：

- **顯示資料列**&#x200B;以檢查建議的目錄 (從頁面上的標題剖析而來，並以錨點連結項目的形式呈現)。
- **預覽**&#x200B;變更前後的比較。
- **標記為已修正**，如果您是在 LLM Optimizer 之外處理機會。
- **忽略**&#x200B;不相關的建議。

建議項目分為&#x200B;**「目前建議」**、**「已修正建議」**&#x200B;和&#x200B;**「已忽略建議」**，這與邊緣最佳化提供的其他機會一致。

### 部署最佳化項目

當您準備好發佈至邊緣時，請選取您要部署的目錄建議。 頁尾會顯示目前選取的項目數量，且通常提供&#x200B;**「標記為已修正」**、**「已忽略建議」**&#x200B;和&#x200B;**「部署最佳化項目」**。

按一下&#x200B;**「部署最佳化項目」**。 **「部署至邊緣」**&#x200B;對話框會列出選取的 URL 和最佳化詳細資訊。 審閱清單，然後選擇&#x200B;**「部署」**&#x200B;或&#x200B;**「取消」**。

成功部署後，**「部署完成」**&#x200B;會確認有多少最佳化項目已上線。 關閉對話框並開啟&#x200B;**「已修正建議」**&#x200B;以驗證狀態。

>[!NOTE]
>
>部署最佳化項目之前，必須先完成邊緣最佳化的上線流程。 如果您尚未上線，按一下&#x200B;**「部署最佳化項目」**，系統便會將您導向上線流程。 如需了解邊緣最佳化的運作方式、支援的內容傳遞網路提供者和上線流程的完整詳細資訊，請參閱[邊緣最佳化](/help/dashboards/optimize-at-edge/overview.md)頁面。

### 已修正建議和檢視即時版本

在&#x200B;**「已修正建議」**&#x200B;中，已部署的 URL 會在狀態欄顯示&#x200B;**「已最佳化」**。 顯示資料列以審閱已部署的目錄、使用&#x200B;**「詳細資料」**&#x200B;取得可用的分析，或按一下&#x200B;**「檢視即時版本」**&#x200B;以唯讀模式開啟為進行驗證而提供的&#x200B;**目前頁面內容** (包括插入的&#x200B;**目錄**)。

當您需要大量復原邊緣變更時，請使用核取方塊選取已最佳化的資料列，然後使用標頭中的&#x200B;**「復原」**。

## 復原

如果您改變主意，可以復原已部署的最佳化項目。 在&#x200B;**「已修正建議」**&#x200B;視圖中，選取您要還原的已最佳化資料列，然後按一下標頭中的&#x200B;**「復原」**。

**「復原」**&#x200B;對話框會列出將要復原的建議，並附上簡短的警告，指出已部署的最佳化項目將會還原。 確認清單，然後按一下&#x200B;**「復原」**&#x200B;或&#x200B;**「取消」**。

操作完成後將會顯示&#x200B;**「已成功復原」**&#x200B;摘要；關閉摘要即可返回儀表板。

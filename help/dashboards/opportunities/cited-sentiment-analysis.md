# 引證情緒分析

當AI系統回答有關您品牌的問題時，它們會依賴一組引用次數最多的URL，也就是在AI產生的回應中經常參考的第三方網頁。 如何在這些頁面上描繪您的品牌，會直接影響AI系統向使用者呈現品牌的方式。

引證情緒分析機會會分析在您的品牌存在感儀表板提示集中針對提示偵測到的最常引證URL。 它會評估這些頁面中的品牌提及、情緒、聲音份額和循環主題。 之後，它會顯示優先建議，以改善AI系統最依賴的內容如何看待您的品牌。

它會顯示四個關鍵量度：

- **已分析頁面** — 已檢查用於品牌提及和情緒的引用網頁數目。
- **略過的頁面** — 無法分析的頁面數目（例如，由於存取限制）。
- **品牌提及（頁面）** — 在分析頁面上提及您品牌的頻率。
- **整體情緒（頁面）** — 跨分析頁面對您品牌的彙總情緒。

>[!NOTE]
>引用的情緒分析目前處於測試階段。 功能與可用性可能會隨著功能的持續開發而改變。

![引用的情緒分析儀表板](/help/dashboards/opportunities/assets/cited-sentiment-overview.png)

## 運作方式

LLM Optimizer會識別在AI產生的回應中，出現在您的品牌存在感儀表板提示集中提示的前幾個引證URL。 它會分析這些頁面以取得品牌提及、情緒、聲音份額和AI引文。 它會比較您的品牌與市場競爭對手的表現、識別週期性主題並產生建議，以解決對AI系統最重要的頁面上的認知缺口。

如果沒有在您的提示集中偵測到提示的引證URL，此機會將不會出現在您的儀表板中。

結果會顯示在兩個索引標籤中： **建議**&#x200B;和&#x200B;**效能**。

## 建議

此標籤顯示改善品牌對引用次數最多的URL之認知度的建議。 建議會整理成三個子標籤： **目前的建議**、**已修正的建議**&#x200B;以及&#x200B;**忽略的建議**。

![建議索引標籤](/help/dashboards/opportunities/assets/cited-sentiment-suggestions.png)

建議表格包含下列各欄：

- **建議** — 建議的改善專案，以解決認知差距。
- **優先順序** — 緊急等級（嚴重、高、Medium、低）。
- **動作專案** — 開啟包含實施建議之特定步驟的面板，包括負責團隊。

展開建議會顯示&#x200B;**AI分析**&#x200B;區段，其中包含：

- **為什麼需要改進** — 說明識別出的認知差距，包括哪些URL沒有充分代表您的品牌和競爭環境。
- **如何改善** — 外聯、內容建立或合作行動的特定指南，以解決差距。
- **預期結果** — 實作建議的預期結果。

## 績效

**效能**&#x200B;索引標籤提供您品牌在引用次數最多的頁面上的表現詳細劃分。 它分為四個區段。

### 市場前景

根據各引用頁面中的提及內容，比較您的品牌和相關品牌以及市場競爭對手的效能。

![市場橫向](/help/dashboards/opportunities/assets/cited-sentiment-landscape.png)

它顯示：

- **個頁面中的品牌提及** — 您的聲音份額與相關品牌和市場競爭者的比較。
- **市場追蹤** — 可篩選的圖表，最多可選取五個競爭者品牌，以比較分析頁面中的聲音份額。

### 情緒分析

使用&#x200B;**情緒分佈**&#x200B;圖表追蹤分析頁面的品牌知覺，該圖表顯示跨頁面有利、中立和不利情緒的百分比劃分。

![情緒分析](/help/dashboards/opportunities/assets/cited-sentiment-distribution.png)

### 頁面

已分析之引用網頁的詳細表格，包含下列欄：

- **頁面** — 分析頁面的URL。
- **品牌提及** — 您的品牌提及數與頁面上的提及總數。
- **聲音份額** — 您品牌相對於所有提及品牌的提及份額。
- **前5名品牌** — 頁面上提及次數最多的品牌。
- **情緒** — 頁面上您品牌的整體情緒。
- **AI引用** — 引用此頁面的AI回應數目。

### 主題

在分析頁面上識別的週期性主題的表格，顯示：

- **主題** — 已識別週期性主題或主旨。
- **品牌提及** — 與主題相關聯的品牌提及數目。
- **情緒** — 與主題相關的整體情緒。

按一下任何主題上的&#x200B;**詳細資料**&#x200B;會開啟一個深入分析，其中包含分析摘要和貢獻來源頁面。

## 常見問題

**為何最常引用的URL對AI 搜尋很重要？**

最常引用的URL是AI系統在產生有關您品牌的回應時最常參照的第三方頁面。 這些頁面上的情緒和框架對於AI代表您品牌的方式有直接和巨大的影響，其影響甚至超過很少被引用的頁面。 改善您品牌在這些特定頁面上的描繪方式，是您為了讓AI可見度而採取的最高槓桿率動作之一。

**為什麼這個機會沒有顯示在我的儀表板中？**

只有在您的品牌存在感儀表板提示集中偵測到引用的URL提示時，才會出現這個機會。 如果沒有為這些提示識別引用的URL，則不會顯示機會。

**略過的頁面是什麼意思？**

略過的頁面會引述為無法分析的URL，通常是因為頁面位於付費牆之後、需要驗證或封鎖自動存取。 這些頁面會計入，但會從情緒和品牌提及分析中排除。

**什麼是聲音份額？**

聲音份額是指您的品牌在指定頁面或所有已分析頁面上，相對於其他所有提及品牌的總體品牌提及百分比。

**什麼是AI引文？**

AI引文顯示特定頁面引用的AI答案數。 較高的AI引文數表示在產生相關主題的回應時，AI系統正積極使用該頁面，使得這些頁面上的情緒對於您品牌的AI表示尤為重要。

**如何識別市場競爭者？**

系統會根據您品牌的產業以及在分析頁面中最常共同提及的品牌，自動識別競爭者。 您也可以手動選取最多5個品牌，以便在市場追蹤圖表中比較。

**分析多久更新一次？**

該分析會反映截至儀表板標題中所顯示日期為止偵測到的引用URL。 實施建議後重新造訪商機，以追蹤情緒和聲音份額中的變更。

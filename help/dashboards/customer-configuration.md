# 客戶組態 {#customer-configuration}

「客戶設定儀表板」是一個功能強大的工具，可讓您深入瞭解品牌在LLM中的可見度。 透過正確設定類別、主題、提示，您可以確保您的品牌處於適當位置，可出現在LLM產生的回應中。 此設定可確保平台根據您的業務內容量身打造深入分析，實現準確的可見度、流量和機會分析。

![客戶設定儀表板](/help/dashboards/assets/customer-config.png)

為了設定LLM Optimizer如何跨不同市場和競爭環境監視和分析您的品牌存在，您可以存取以下標籤：

* [類別](#categories)
* [其他追蹤](#others-tracking)
* [品牌別名](#brand-aliases)
* [資料分析](#data-insights)
* [CDN 設定](#agentic-cdn)

>[!IMPORTANT]
>
> 如需有關如何設定類別、主題、提示的詳細資訊，請參閱[設定類別、主題、提示的最佳實務](/help/overview/best-practices-topics-prompts.md)頁面。

## 類別 {#categories}

從類別頁標，您可以定義要追蹤的業務類別或產品線，並將其與特定區域建立關聯。 總體而言，類別標籤與此頁面上的幾乎所有其他自訂都有關，因為類別會出現在其他自訂專案（其他追蹤、別名等）的類別欄位中。 若要新增類別：

1. 按一下&#x200B;**新增**&#x200B;按鈕。
2. 在新設定視窗中，新增&#x200B;**類別名稱**。
3. 自訂將監視類別的&#x200B;**關聯區域**。
4. 按一下&#x200B;**儲存**，新類別就會出現在類別清單中。

新增類別不會自動產生主題和提示，這些主題和提示需要從[資料深入分析](#data-insights)索引標籤手動新增。

若要刪除類別，請按一下類別清單中的刪除圖示。 請小心，因為&#x200B;**刪除類別也會刪除連結至該特定類別的相關專案** （如品牌別名）。

## 其他追蹤 {#others-tracking}

透過使用此標籤，您可以追蹤不同類別和區域中他人與您品牌的關係。 監控其在您市場區段中的現況和效能。 若要自訂追蹤：

1. 按一下&#x200B;**新增**&#x200B;按鈕。
2. 在新的設定視窗中，選取&#x200B;**類別**。 先前建立的類別將顯示在這裡。
3. 新增其他人的名稱。
4. 視需要自訂其他別名與網域。
5. 按一下&#x200B;**儲存**。

若要刪除清單上的專案，請按一下刪除圖示。

## 品牌別名 {#brand-aliases}

透過使用品牌別名，您可以設定應跨不同類別和區域追蹤的替代名稱和品牌變體。 這可確保全面監控所有品牌提及。 若要新增品牌別名：

1. 按一下&#x200B;**新增**&#x200B;按鈕。
2. 在新的設定視窗中，選取&#x200B;**類別**。 先前建立的類別將顯示在這裡。
3. 選取要監視別名的&#x200B;**區域**。
4. 新增品牌別名。
5. 按一下&#x200B;**儲存**，品牌別名就會出現在清單中。

若要刪除品牌別名，請按一下別名清單中的刪除圖示。

## 資料分析 {#data-insights}

您可以從此標籤檢閱、管理及自訂提示。 您可以上傳[Brand Presence資料深入分析](/help/dashboards/brand-presence.md#data-insights) .csv，清單中會填入該分析的提示和主題。 您也可以視需要刪除、修改和新增主題及其相關提示。

若要匯入資料深入分析.csv檔案，您必須先從Brand Presence儀表板匯出檔案。 請參閱[資料深入分析](/help/dashboards/brand-presence.md#data-insights)章節以瞭解如何操作。 取得檔案後：

1. 在儀表板上，按一下&#x200B;**上傳CSV**。
2. 在匯入資料深入分析視窗上，拖放或手動選擇檔案。
3. 按一下&#x200B;**上傳資料**。

您也可以從&#x200B;**匯入資料分析**&#x200B;視窗下載範本，以建立新的CSV檔案。 取得範本後，請開啟範本，並以新行輸入主題及其相關的提示、類別和區域。

此外，您也可以在CSV檔案以外單獨將主題/提示新增至清單。 若要完成此操作，在控制面板上，您需要：

1. 按一下&#x200B;**新增主題**&#x200B;按鈕。
2. 在新的設定視窗中，選取&#x200B;**類別**。 先前建立的類別將顯示在這裡。
3. 輸入主題名稱。
4. 新增提示文字。
5. 選取地區。
6. 按一下&#x200B;**新增提示**，含有提示的主題就會出現在清單中。

>[!NOTE]
>處理完成前，新新增的提示不會出現在品牌顯示中。

在清單中，您可以按一下每個主題，相關的提示就會顯示。若要刪除主題及其相關的提示，請按一下清單中的刪除圖示。

## CDN 設定 {#cdn-configuration}

您可以在此標籤設定CDN串流，讓Adobe LLM Optimizer分析您的CDN資料。 此資料將用於加強儀表板（例如代理和反向連結流量），提供流量模式、績效量度和最佳化機會的深入分析。 若要加入CDN提供者，請按一下&#x200B;**加入CDN**。

![客戶組態CDN](/help/overview/assets/cc-cdn.png)

在&#x200B;**內建CDN提供者**&#x200B;視窗上：

1. 選取您的CDN提供者。
2. 按一下&#x200B;**內建**&#x200B;以啟用記錄轉送。

如果您選取&#x200B;**其他**，則必須連絡llmo-now@adobe.com以取得協助。

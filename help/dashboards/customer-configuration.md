# 客戶組態

「客戶設定」可讓您定義在平台內如何監控及分析您的品牌。 您可以自訂類別（例如業務單位或產品線）、追蹤競爭者，以及新增品牌提及別名，以擷取不同提示中您品牌的所有變體。 此設定可確保平台根據您的業務內容量身打造深入分析，實現準確的可見度、流量和機會分析。

![競爭引文的趨勢URL](/help/dashboards/assets/customer-config.png)

為了設定LLM Optimizer如何跨不同市場和競爭環境監視和分析您的品牌存在，您可以存取以下標籤：

* [類別](#categories)
* [競爭者追蹤](#competitor-tracking)
* [品牌別名](#brand-aliases)
* [資料分析](#data-insights)
* [代理CDN](#agentic-cdn)

## 類別 {#categories}

從類別頁標，您可以定義要追蹤的業務類別或產品線，並將其與特定區域建立關聯。 整體而言，類別標籤會與本頁面上的其他每個自訂專案相關，因為其他自訂專案（競爭者追蹤、別名等）的類別欄位中會顯示類別。若要新增類別：

1. 按一下&#x200B;**新增**&#x200B;按鈕。
2. 在新設定視窗中，新增&#x200B;**類別名稱**。
3. 自訂將監視類別的&#x200B;**關聯區域**。
4. 按一下&#x200B;**儲存**，新類別就會出現在類別清單中。

新增類別不會自動產生主題和提示，這些主題和提示需要從[資料深入分析](#data-insights)索引標籤手動新增。

若要刪除類別，請按一下類別清單中的刪除圖示。 請小心，因為&#x200B;**刪除類別也會刪除關聯的專案**，例如您可能已設定的競爭者或連結至該特定類別的品牌別名。

## 競爭者追蹤 {#competitor-tracking}

透過使用競爭者追蹤，您可以跨不同類別和區域追蹤與您的品牌相關的競爭者提及方式。 監控其在您市場區段中的現況和效能。 若要自訂競爭者追蹤：

1. 若要新增競爭者，請按一下[新增]按鈕。**&#x200B;**
2. 在新的設定視窗中，選取&#x200B;**類別**。 先前建立的類別將顯示在這裡。
3. 新增競爭者的名稱。
4. 視需要自訂競爭者的「別名」和「網域」。
5. 按一下「儲存」**&#x200B;**，新的競爭者就會出現在競爭者清單中。

若要刪除競爭者，請按一下競爭者清單中的刪除圖示。

## 品牌別名 {#brand-aliases}

透過使用品牌別名，您可以設定應跨不同類別和區域追蹤的替代名稱和品牌變體。 這可確保全面監控所有品牌提及。 若要新增品牌別名：

1. 按一下&#x200B;**新增**&#x200B;按鈕。
2. 在新的設定視窗中，選取&#x200B;**類別**。 先前建立的類別將顯示在這裡。
3. 選取要監視別名的&#x200B;**區域**。
4. 新增品牌別名。
5. 按一下&#x200B;**儲存**，品牌別名就會出現在清單中。

若要刪除品牌別名，請按一下別名清單中的刪除圖示。

## 資料分析 {#data-insights}

您可以從此標籤檢閱、管理及自訂提示。 您可以上傳[Brand Presence資料深入分析](/help/dashboards/brand-presence.md#data-insights) .csv，清單中會填入該分析的提示和主題。 您也可以視需要刪除、修改和新增主題及其相關提示。

若要匯入資料深入分析.csv檔案，您必須先從Brand Presence儀表板匯出檔案。 請參閱[資料深入分析](/help/dashboards/brand-presence.md#data-insights)章節以瞭解如何操作。 取得檔案後：

1. 在儀表板上，按一下&#x200B;**上傳CSV**。
2. 在匯入資料深入分析視窗上，拖放或手動選擇檔案。
3. 按一下&#x200B;**上傳資料**。

您也可以從&#x200B;**匯入資料分析**&#x200B;視窗下載範本，以建立新的CSV檔案。 取得範本後，請開啟範本，並以新行輸入主題及其相關的提示、類別和區域。

此外，您也可以在CSV檔案以外單獨將主題/提示新增至清單。 若要完成此操作，在控制面板上，您需要：

1. 按一下&#x200B;**新增主題**&#x200B;按鈕。
2. 在新的設定視窗中，選取&#x200B;**類別**。 先前建立的類別將顯示在這裡。
3. 輸入主題名稱。
4. 新增提示文字。
5. 選取地區。
6. 按一下&#x200B;**新增提示**，含有提示的主題就會出現在清單中。

>[!NOTE]
>處理完成前，新新增的提示不會出現在品牌顯示中。

在清單中，您可以按一下每個主題，相關的提示就會顯示。若要刪除主題及其相關的提示，請按一下清單中的刪除圖示。

# URL檢查器

「URL檢測器」可協助您分析網域上的特定頁面在AI搜尋中的執行情形。 它結合了URL層級的可見度、一般流量和參照資料，為您提供哪些URL被引用以及它們在回應中出現的頻率的精細檢視。 **SR-NEEDS CDN？**

![URL檢測器](/help/dashboards/assets/url-insp.png)

## 篩選器

在頁面頂端，您可以套用篩選條件來調整檢視。 您選擇的篩選器將影響儀表板上呈現的&#x200B;**所有**&#x200B;區段。 您可以自訂下列專案：

* **日期範圍** — 選取所顯示資料的時間範圍。 例如最近4週。 您也可選擇選取&#x200B;**自訂周數**&#x200B;選項，以自訂時段。
* **類別** — 依類別篩選顯示的結果。
* **平台** — 選擇要分析的AI引擎。
* **管道** — 在贏取、競爭者和社交等管道之間篩選。
* **地區** — 依地理位置篩選結果。 並非所有區域在啟動時都可使用。
選取想要的篩選器後，按一下**套用篩選器**，將選取專案套用至儀表板。

## 總覽量度

「URL檢測器」提供數個概觀量度，讓您能夠快速評估頁面在AI搜尋中的表現。 提供下列量度：

* **擁有引文的唯一提示** — 擁有引文的唯一人工智慧提示總數。(**SR — 更多詳細資料？ 擁有哪些引文**)
* **唯一提示總數** — 唯一AI提示總數。
* **唯一被引用的URL** — 被引用的唯一擁有的URL數目。
* **引證的總次數** — 擁有的URL在AI產生的答案中被引證的總次數。
* **代理程式點選總數** — 您URL上來自AI代理程式的點選總數。
* **來自LLM的轉介點選** — 從AI產生的回應導向至您URL的點選總數。

每個概觀量度的趨勢指標會顯示這些值與上一個時段相比如何隨著時間改變。

## 您引用的URL

引證URL的檢視會列出在AI產生的答案中引證的來自您品牌的所有URL，以及支援的量度。 資料表格也有搜尋欄位，可快速存取特定URL。 提供下列量度：

* **URL** — 分析過的URL
* **引證次數** — 在AI產生的答案中引證URL的次數。
* **在**&#x200B;中引用的提示 — 引用URL的唯一AI提示數目。
* **類別** — 與URL相關聯的產品類別或主題。
* **地區** — 引用URL的地理區域。
* **代理點選** - URL上來自AI代理的點選總數。
* **轉介點選** — 從AI產生的回答導向到URL的點選數。

## 競爭引文的熱門URL

競爭引文檢視的熱門URL會強調目前被引用於品牌相關答案的外部URL，衡量誰是您分享空間中的成功引文。 資料表格具有搜尋欄位，可快速存取特定URL。 此外，您可以使用&#x200B;**匯出**&#x200B;選項來下載資料表.csv，並與您的團隊共用深入分析或將資料表納入執行報告中。

![競爭引文的趨勢URL](/help/dashboards/assets/trend-url.png)

提供下列量度：

* **URL** — 分析過的URL
* **內容型別** — 內容型別（擁有、社交、盈餘、競爭者）。
* **引證次數** — 在AI產生的答案中引證URL的次數。
* **在**&#x200B;中引用的提示 — 引用URL的唯一AI提示數目。
* **類別** — 與URL相關聯的產品類別或主題。
* **地區** — 引用URL的地理區域。

將滑鼠游標停留在每個URL上時，每個URL都有一個&#x200B;**詳細資料**&#x200B;按鈕。 按一下按鈕將顯示另一個視窗，其中包含更多詳細資訊。

# URL 檢查器

URL 檢查器可協助您分析網域上的特定頁面在 AI 搜尋中的表現。此檢查器將 URL 層級的能見度、代理式流量和轉介資料結合，讓您深入細節，了解哪些 URL 被引用以及這些 URL 出現在回答中的頻率。

![URL 檢查器](/help/dashboards/assets/url-insp.png)

## 篩選器

在頁面頂端，您可以套用篩選器來精確調整視圖。您選擇的篩選器將影響儀表板上&#x200B;**所有**&#x200B;區段。您可以自訂下列項目：

* **日期範圍**：選取所顯示資料的時間範圍。例如最近 4 週。您也可選取「**自訂週數**」選項來自訂時段。
* **類別**：依類別篩選所顯示的結果。
* **平台**：選擇要分析的 AI 引擎。
* **頁面內容類型**：依內容類型進行篩選。
* **區域**：依地理位置篩選結果。並非所有區域在啟動時皆會開放使用。

選取想要的篩選器後，按一下「**套用篩選器**」，將選取項目套用至儀表板。

## 概觀量度

URL 檢查器提供數個概觀量度，讓您能夠快速評估相關頁面在 AI 搜尋中的表現。所提供的量度如下：

* **包含自有品牌引用的不重複提示**：包含自有品牌引用的不重複 AI 提示總數。
* **不重複提示總數**：不重複 AI 提示的總數。
* **不重複的被引用 URL**：不重複的被引用自有 URL 的數量。
* **總引用次數**：自有 URL 在 AI 生成回答中被引用的總次數。
* **代理式點擊總數**：AI 代理對您的 URL 的點擊總數。
* **來自 LLM 的轉介點擊數**：從 AI 生成回答引導至您的 URL 的點擊總數。

每個概觀量度的趨勢指標會顯示這些值在一段時間內與前一時段相比的變化。

## 您的被引用 URL

被引用 URL 視圖會列出您的品牌之下在 AI 生成回答中被引用的所有 URL，並附上佐證量度。兩個表格皆具有方便快速存取各個主題的搜尋欄位，而您只要按一下「**設定欄**」按鈕即可自訂要顯示的量度。此外，您可以使用「**匯出**」選項來下載 .csv 格式的表格，並與您的團隊共用洞察或將表格納入執行報告中。

![被引用的 URL](/help/dashboards/assets/cited-urls.png)

所提供的量度如下：

* **URL**：所分析的 URL。
* **引用次數**：URL 在 AI 生成回答中被引用的次數。
* **進行引用的提示**：引用 URL 的不重複 AI 提示數。
* **類別**：與 URL 相關聯的產品類別或主題。
* **區域**：引用 URL 的地理區域。
* **代理式點擊數**：AI 代理對該 URL 的點擊總數。
* **轉介點擊數**：從 AI 生成回答引導至 URL 的點擊數。

## 競爭引用機會的熱門 URL

「競爭引用機會的熱門 URL」視圖，會特別標示目前在與您的品牌相關的回答中引用的外部 URL，衡量何者在您的領域中贏得引用。資料表格具有方便快速存取特定 URL 的搜尋欄位。此外，您可以使用「**匯出**」選項來下載 .csv 格式的表格，並與您的團隊共用洞察或將表格納入執行報告中。

![競爭引用機會的熱門 URL](/help/dashboards/assets/trend-url.png)

所提供的量度如下：

* **URL**：所分析的 URL。
* **內容類型**：內容的類型 (自有、社交媒體、爭取、其他)。
* **引用次數**：URL 在 AI 生成回答中被引用的次數。
* **進行引用的提示**：引用 URL 的不重複 AI 提示數。
* **類別**：與 URL 相關聯的產品類別或主題。
* **區域**：引用 URL 的地理區域。

### 詳細資料視窗

在引用和趨勢分析視圖中，URL 在每列的末尾都有「**詳細資料**」按鈕。按一下按鈕將顯示包含其他詳細資料的另一個視窗。詳細資料視窗顯示 URL 引用頻率、<!--the sentiment of AI responses where it is mentioned,-->出現 URL 的主題和提示，以及 (自有 URL 的) 代理式和轉介流量在一段時間內的趨勢。

![詳細資料視窗](/help/dashboards/assets/details-url.png)

# 協作

您可以透過協作邀請團隊成員加入貴組織的 LLM Optimizer 工作環境，讓每個人都可以同一位置存取洞察和分析。您可以透過新增同事，確保行銷、SEO、公關和技術團隊能夠無縫合作，共同改善品牌能見度、分享發現，以及針對各項機會採取協同行動。

## 如何邀請團隊成員

您可以邀請團隊成員，只要將成員的電子郵件新增至「**電子郵件地址**」欄位，然後按一下「**傳送邀請**」即可。如果他們接受邀請，就會加入貴組織的 LLM Optimizer 工作環境。

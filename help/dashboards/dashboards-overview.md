# 控制面板概觀

歡迎使用儀表板概觀。 LLM-Optimizer中的儀表板可協助行銷人員追蹤各個AI搜尋平台的品牌可見度和情緒、找出最佳化機會，以及實作規範性變更以改善引文和效能。

重要檢視包括品牌存在、代理和反向連結流量、URL檢測器、機會識別、Collaboration和客戶設定。
